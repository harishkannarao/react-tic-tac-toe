# Tic Tac Toe - React

## Tools Required
* NodeJs 12: `node --version`
* Npm 6: `npm --version`
* Npx 6: `npx --version`
* Yarn 1.22: `yarn --version`

## Project creation

    npx create-react-app

## Commands

### Install node modules

    yarn install

### Start application

    yarn start

    yarn start:no-browser

### Build application

    yarn build

### Run the build distribution

    yarn serve

### Run the unit and component tests

    yarn test

    yarn test:ci
